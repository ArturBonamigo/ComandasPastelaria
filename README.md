# 🍽️ Comandas Pastelaria

🎓 **Projeto Acadêmico**

Sistema Full Stack desenvolvido para gerenciamento de comandas de uma pastelaria, permitindo o controle de clientes, produtos, funcionários, comandas e caixa por meio de uma API REST e uma interface web.

---

## 🧩 Funcionalidades

- Autenticação de usuários
- Cadastro de clientes
- Cadastro de produtos
- Cadastro de funcionários
- Abertura de comandas
- Registro de consumo
- Gerenciamento de recebimentos
- Controle de caixa
- Auditoria de operações
- Interface web integrada à API

---
